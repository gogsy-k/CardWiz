💳 CardWiz — Privacy Policy

Last updated: 12 June 2026
TL;DR — By default aapka saara data sirf aapke device pe rehta hai. Agar aap Google se sign in karte ho, ek account banta hai aur Cloud Sync (default ON, jise aap off kar sakte ho) aapke cards ki limited details — card type, nickname, last-4 (poora number nahi), due-date — hamare secure server pe sync karta hai, taaki kisi bhi browser pe login karke wahi cards mil sakein. Koi tracking/ads nahi. Poora card number / CVV kabhi nahi.
Hum ye KABHI collect/store nahi karte:
  • Poora credit/debit card number
  • CVV
  • Net-banking / bank login credentials
  • OTP ya koi payment authentication

1. Kaun-sa data store hota hai (sirf device pe)

Aap jo add karte ho, woh browser ke chrome.storage.local mein save hota hai — sirf aapke is device/browser pe:

  • Aapke cards ka type/naam (jaise "HDFC Millennia") aur optional nickname
  • Optional last 4 digits (sirf pehchaan ke liye — poora number nahi)
  • Bill due date (mahine ka din) aur reminder preference
  • Cap usage jo aap manually log karte ho ("✓ Use kiya")
  • Premium on/off setting

Signed out ho to ye data kabhi aapke device se bahar nahi jaata. Signed in + Cloud Sync ON ho to upar batayi limited card details (last-4 tak, poora number nahi) hamare server pe sync hoti hain — niche section 2 dekho.

2. Account & Cloud Sync (optional)

Agar aap Google se sign in karte ho, hum ek account banakar secure server pe store karte hain: aapka email, naam, Google account id, aur plan (free/premium). Sign-in optional hai — bina iske bhi extension poora kaam karta hai.

Cloud Sync (default ON, "More" tab se OFF kar sakte ho): signed-in hone pe aapke cards ki limited details server pe sync hoti hain taaki kisi bhi browser pe login karke wahi cards milein. Sync hota hai sirf:

  • Card type/naam aur nickname
  • Last 4 digits (poora number nahi)
  • Due date + reminder preference

Server pe bhi poora card number / CVV / bank login KABHI nahi jaata. Cloud Sync OFF karoge to cards sirf is device pe rehte hain.

3. Shopping sites pe kya hota hai

Amazon.in, Flipkart.com aur Myntra.com ke cart/checkout pages pe extension read-only taur pe ye padhta hai:

  • Order ka total amount
  • Page pe dikhaye gaye bank offers / No-Cost-EMI text

Ye sab aapke browser ke andar hi process hota hai — best card recommend karne ke liye. Ye jaankari kahin transmit nahi hoti. Extension koi form fill nahi karta, koi card number enter nahi karta.

4. Notifications

Bill reminders ke liye browser notifications use hoti hain. Ye aapke device pe locally schedule hoti hain (chrome.alarms) — koi external push service nahi.

5. Affiliate links

"Buy via our link" dabane pe, link mein hamara affiliate tag add hota hai. Agar aap kuch khareedte ho, to hame chhota commission milta hai — aapko koi extra cost nahi, price wahi rehta hai. Affiliate network (jaise Amazon Associates) ko sirf standard click-referral jaati hai; hum aapki koi personal jaankari unke saath share nahi karte.

6. Permissions kyun chahiye

  • storage — aapke cards/settings device pe save karne ke liye
  • alarms — bill reminder check schedule karne ke liye
  • notifications — bill due reminders dikhane ke liye
  • identity — optional Google sign-in (account + cloud sync) ke liye
  • Amazon/Flipkart/Myntra access — checkout pe best card recommend karne ke liye (read-only)

7. Data delete kaise karein

Extension uninstall karte hi saara local data delete ho jaata hai. Aap "Clear" / "Caps reset" buttons se bhi data hata sakte ho. Agar aap signed in the, to synced cards hatane ke liye saare cards delete kar do (empty set server pe chala jaata hai), ya account/data deletion ke liye humein likho.

8. Bachchon ke liye

Ye extension financial tool hai, 18+ users ke liye intended hai.
